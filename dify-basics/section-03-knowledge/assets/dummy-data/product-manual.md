# FlowMaster ユーザーマニュアル

## 製品概要

FlowMasterは、AI技術を活用した次世代のプロジェクト管理ツールです。チームのコラボレーションを促進し、プロジェクトの進捗を可視化し、タスクの自動割り当てを実現します。

### 主な機能

- **AIタスク自動割り当て**: チームメンバーのスキルと負荷を分析し、最適なタスク配分を提案
- **進捗予測**: プロジェクトの完了予定日をAIが予測
- **リアルタイムコラボレーション**: チーム全員が同時に作業できる共同編集機能
- **スマート通知**: 重要な更新のみを通知し、ノイズを削減
- **統合API**: SlackやGitHubなど主要なツールと連携可能

## 第1章: セットアップ

### 1.1 システム要件

#### 推奨環境
- **OS**: Windows 10以降、macOS 11以降、Ubuntu 20.04以降
- **ブラウザ**: Chrome 90+、Firefox 88+、Safari 14+、Edge 90+
- **メモリ**: 4GB以上
- **ディスク空き容量**: 500MB以上
- **インターネット接続**: 常時接続が必要

#### モバイル環境
- **iOS**: iOS 14以降
- **Android**: Android 9以降

### 1.2 アカウント作成

1. [https://flowmaster.tech](https://flowmaster.tech) にアクセス
2. 「無料で始める」ボタンをクリック
3. メールアドレスとパスワードを入力
4. 受信したメールのリンクをクリックして認証
5. プロフィール情報を入力して完了

### 1.3 初期設定

#### ワークスペースの作成

1. ログイン後、「新規ワークスペース」をクリック
2. ワークスペース名を入力（例：「マーケティングチーム」）
3. チームメンバーのメールアドレスを追加
4. 招待メールを送信

#### プロジェクトの作成

1. ワークスペースを選択
2. 「新規プロジェクト」ボタンをクリック
3. 以下の情報を入力：
   - プロジェクト名
   - 説明
   - 開始日と終了日
   - プロジェクトテンプレート（オプション）
4. 「作成」をクリック

## 第2章: 基本操作

### 2.1 タスクの作成

#### 手動作成

1. プロジェクト画面で「+タスク」をクリック
2. タスク名を入力
3. 詳細情報を設定：
   - **優先度**: 低・中・高・緊急
   - **期限**: カレンダーから選択
   - **担当者**: ドロップダウンから選択
   - **タグ**: 関連するタグを追加
   - **説明**: Markdown形式で詳細を記述
4. 「保存」をクリック

#### AI自動作成

1. プロジェクト説明を入力
2. 「AIにタスクを提案させる」をクリック
3. AIが生成したタスクリストを確認
4. 必要に応じて編集
5. 「すべて追加」で一括作成

### 2.2 タスクの管理

#### ステータスの変更

タスクカードをドラッグ&ドロップで以下のカラム間を移動：
- **To Do**: 未着手
- **In Progress**: 作業中
- **Review**: レビュー待ち
- **Done**: 完了

#### タスクの編集

1. タスクカードをクリック
2. 詳細パネルが表示される
3. 必要な項目を編集
4. 自動保存される

#### サブタスクの追加

1. タスクの詳細画面を開く
2. 「サブタスクを追加」をクリック
3. サブタスク名を入力
4. Enterキーで次のサブタスクを追加

### 2.3 ビューの切り替え

FlowMasterは4つのビューを提供：

#### カンバンビュー
- デフォルトのビュー
- ステータスごとにタスクを整理
- ドラッグ&ドロップで簡単に移動

#### リストビュー
- タスクを一覧表示
- ソートとフィルタリングが可能
- CSV形式でエクスポート可能

#### カレンダービュー
- タスクを期限ごとにカレンダー上に表示
- 日・週・月単位で表示切替
- 外部カレンダーと同期可能

#### ガントチャート
- プロジェクトのタイムラインを視覚化
- タスクの依存関係を表示
- クリティカルパスをハイライト

## 第3章: 高度な機能

### 3.1 AI機能の活用

#### タスク自動割り当て

1. プロジェクト設定で「AI自動割り当て」を有効化
2. チームメンバーのスキルセットを登録
3. 新規タスク作成時、AIが最適な担当者を提案
4. 提案を承認または手動で変更

#### 進捗予測

1. ダッシュボードで「進捗予測」タブを開く
2. AIが過去のデータから完了予定日を計算
3. リスク要因を分析して表示
4. 予測に基づいてアラートを設定可能

### 3.2 通知設定

#### 通知のカスタマイズ

1. 右上のアイコン → 「設定」をクリック
2. 「通知」タブを選択
3. 以下の項目を設定：
   - タスクの割り当て通知
   - 期限のリマインダー（1日前、1時間前など）
   - コメントの通知
   - プロジェクトの更新通知
4. 「保存」をクリック

#### スマート通知

AIが重要度を判断し、優先度の高い通知のみを表示：
- **即時通知**: 緊急のタスクやメンション
- **まとめ通知**: 低優先度の更新は1日1回まとめて通知
- **サイレント通知**: 通知センターのみに表示

### 3.3 統合機能

#### Slack連携

1. 「設定」→「統合」→「Slack」を選択
2. 「Slackに接続」をクリック
3. ワークスペースを選択して認証
4. 通知先のチャンネルを設定
5. 通知ルールをカスタマイズ

#### GitHub連携

1. 「設定」→「統合」→「GitHub」を選択
2. 「GitHubに接続」をクリック
3. リポジトリへのアクセスを許可
4. タスクとIssueを連携
5. コミットとタスクを自動リンク

#### API利用

1. 「設定」→「開発者」→「APIキー」を選択
2. 「新規APIキー」をクリック
3. スコープ（読み取り・書き込み）を設定
4. APIキーをコピーして安全に保管
5. [API ドキュメント](https://docs.flowmaster.tech/api)を参照

## 第4章: トラブルシューティング

### 4.1 よくある問題と解決方法

#### Q1: タスクが保存されない

**原因**:
- インターネット接続が不安定
- ブラウザのキャッシュが破損

**解決方法**:
1. インターネット接続を確認
2. ページを再読み込み（Ctrl+R / Cmd+R）
3. ブラウザのキャッシュをクリア
4. 別のブラウザで試す

#### Q2: AI機能が動作しない

**原因**:
- AI機能が有効になっていない
- プランがAI機能に対応していない

**解決方法**:
1. 「設定」→「AI機能」で有効化を確認
2. プランをアップグレード（Proプラン以上が必要）
3. サポートに問い合わせ

#### Q3: 通知が届かない

**原因**:
- 通知設定が無効
- メールがスパムフォルダに振り分けられている
- ブラウザの通知権限が拒否されている

**解決方法**:
1. 「設定」→「通知」で設定を確認
2. メールのスパムフォルダを確認
3. ブラウザの通知権限を許可
4. メール受信ルールを確認

#### Q4: パスワードを忘れた

**解決方法**:
1. ログイン画面の「パスワードを忘れた」をクリック
2. 登録メールアドレスを入力
3. 受信したメールのリンクをクリック
4. 新しいパスワードを設定
5. ログインして確認

#### Q5: チームメンバーが招待を受け取れない

**原因**:
- メールアドレスが間違っている
- 招待メールがスパムに振り分けられている
- メールサーバーがブロックしている

**解決方法**:
1. メールアドレスを再確認
2. スパムフォルダを確認するよう依頼
3. 「設定」→「チーム」から再招待
4. 招待リンクを直接共有

### 4.2 エラーコード一覧

| エラーコード | 説明 | 対処方法 |
|------------|------|---------|
| ERR_001 | ネットワーク接続エラー | インターネット接続を確認 |
| ERR_002 | 認証エラー | ログアウトして再ログイン |
| ERR_003 | 権限不足 | 管理者に権限付与を依頼 |
| ERR_004 | ストレージ上限 | プランをアップグレード |
| ERR_005 | API制限超過 | 1時間待ってから再試行 |

### 4.3 パフォーマンス最適化

#### ブラウザのパフォーマンス改善

1. 不要なタブを閉じる
2. ブラウザ拡張機能を無効化
3. ハードウェアアクセラレーションを有効化
4. キャッシュを定期的にクリア

#### 大規模プロジェクトの管理

- タスク数が1000を超える場合は、プロジェクトを分割することを推奨
- アーカイブ機能を使って完了したタスクを整理
- フィルタとタグを活用して表示タスクを絞り込む

## 第5章: セキュリティとプライバシー

### 5.1 データセキュリティ

- **暗号化**: すべてのデータはAES-256で暗号化
- **バックアップ**: 1日3回自動バックアップ
- **データセンター**: ISO 27001認証取得済み
- **アクセス制御**: 役割ベースのアクセス管理

### 5.2 二段階認証（2FA）

1. 「設定」→「セキュリティ」→「二段階認証」を選択
2. 「有効化」をクリック
3. QRコードをAuthenticatorアプリでスキャン
4. 表示された6桁のコードを入力
5. バックアップコードを安全に保管

### 5.3 データのエクスポート

1. 「設定」→「データ」→「エクスポート」を選択
2. エクスポート形式を選択（JSON、CSV、PDF）
3. 対象期間を指定
4. 「エクスポート開始」をクリック
5. 完了後、ダウンロードリンクをメールで受信
