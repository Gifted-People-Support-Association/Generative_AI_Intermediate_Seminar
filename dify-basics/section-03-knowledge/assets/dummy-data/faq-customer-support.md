## アカウント・認証

### Q1: アカウントの作成方法を教えてください

**A**: FlowMasterのアカウントは以下の手順で作成できます。

1. [https://flowmaster.tech](https://flowmaster.tech) にアクセス
2. 「無料で始める」ボタンをクリック
3. メールアドレスとパスワードを入力
4. 受信したメールのリンクをクリックして認証
5. プロフィール情報を入力して完了

アカウント作成は無料で、クレジットカードの登録は不要です。

### Q2: パスワードを忘れてしまいました

**A**: パスワードのリセット手順は以下の通りです。

1. ログイン画面の「パスワードを忘れた」をクリック
2. 登録メールアドレスを入力
3. 受信したメールのリンクをクリック
4. 新しいパスワードを設定
5. ログインして確認

メールが届かない場合は、スパムフォルダを確認してください。それでも解決しない場合は、support@flowmaster.tech までお問い合わせください。

### Q3: 二段階認証（2FA）を設定したい

**A**: セキュリティ強化のため、二段階認証の設定を推奨しています。

**設定手順**:
1. 「設定」→「セキュリティ」→「二段階認証」を選択
2. 「有効化」をクリック
3. QRコードをAuthenticatorアプリでスキャン（Google Authenticator、Authyなど）
4. 表示された6桁のコードを入力
5. バックアップコードを安全に保管

二段階認証を有効にすると、ログイン時にパスワードに加えて6桁のコードの入力が必要になります。

### Q4: メールアドレスを変更したい

**A**: メールアドレスの変更手順は以下の通りです。

1. 「設定」→「アカウント」→「メールアドレス」を選択
2. 新しいメールアドレスを入力
3. 確認メールが新旧両方のアドレスに送信される
4. 両方のメールのリンクをクリックして確認
5. 変更完了

セキュリティ上、変更には両方のメールアドレスでの確認が必要です。

### Q5: アカウントを削除したい

**A**: アカウントを削除すると、すべてのデータが完全に削除されます。

**削除手順**:
1. 「設定」→「アカウント」→「アカウント削除」を選択
2. 削除の理由を選択（任意）
3. パスワードを入力して確認
4. 「完全に削除」をクリック

**注意事項**:
- 削除されたデータは復元できません
- 削除前にデータをエクスポートすることを推奨します
- 有料プランの場合、残りの期間は日割り計算で返金されます

## 支払い・プラン

### Q6: 料金プランの違いは何ですか？

**A**: FlowMasterは3つのプランを提供しています。

| 機能 | Free | Pro | Enterprise |
|------|------|-----|-----------|
| ユーザー数 | 最大5名 | 最大50名 | 無制限 |
| プロジェクト数 | 3個 | 無制限 | 無制限 |
| ストレージ | 1GB | 10GB | 無制限 |
| AI機能 | - | ✓ | ✓ |
| 統合機能 | 基本のみ | すべて | すべて |
| サポート | メール | 優先メール | 電話・専任担当 |
| 価格 | 無料 | ¥1,200/月 | お問い合わせ |

詳細は [https://flowmaster.tech/pricing](https://flowmaster.tech/pricing) をご覧ください。

### Q7: 無料プランから有料プランにアップグレードしたい

**A**: アップグレードは簡単に行えます。

**手順**:
1. 「設定」→「プラン」を選択
2. 希望するプランを選択（Pro or Enterprise）
3. 支払い方法を入力（クレジットカード or 請求書）
4. 「アップグレード」をクリック
5. 即座に新機能が利用可能

**支払い方法**:
- クレジットカード（Visa、MasterCard、JCB、American Express）
- 請求書払い（Enterpriseプランのみ、法人のみ）

### Q8: 途中でプランをダウングレードできますか？

**A**: はい、いつでもダウングレード可能です。

**手順**:
1. 「設定」→「プラン」→「プラン変更」を選択
2. 希望するプランを選択
3. 「ダウングレード」をクリック

**注意事項**:
- ダウングレードは次の請求サイクルから適用されます
- 現在の期間中は既存のプランの機能が利用可能です
- ダウングレード後、プラン制限を超えるデータ（プロジェクト数、ストレージ）は削除が必要になる場合があります

### Q9: 領収書を発行してほしい

**A**: 領収書は自動で発行されます。

**取得方法**:
1. 「設定」→「支払い」→「請求履歴」を選択
2. 該当する請求をクリック
3. 「領収書をダウンロード」をクリック

PDFファイルがダウンロードされます。社名や宛名の変更が必要な場合は、support@flowmaster.tech までお問い合わせください。

### Q10: 返金はできますか？

**A**: 30日間の返金保証があります。

**条件**:
- 初回購入から30日以内
- 理由を問わず全額返金

**申請方法**:
1. support@flowmaster.tech に返金希望のメールを送信
2. 申請から5営業日以内に返金処理
3. クレジットカードの場合、カード会社の処理に5-10営業日

## 基本操作

### Q11: タスクの作成方法を教えてください

**A**: タスクは2つの方法で作成できます。

**方法1: 手動作成**
1. プロジェクト画面で「+タスク」をクリック
2. タスク名を入力
3. 詳細情報を設定（優先度、期限、担当者など）
4. 「保存」をクリック

**方法2: AI自動作成**（Proプラン以上）
1. プロジェクト説明を入力
2. 「AIにタスクを提案させる」をクリック
3. AIが生成したタスクリストを確認
4. 必要に応じて編集
5. 「すべて追加」で一括作成

### Q12: タスクのステータスを変更するには？

**A**: タスクカードをドラッグ&ドロップで移動できます。

**ステータス**:
- **To Do**: 未着手
- **In Progress**: 作業中
- **Review**: レビュー待ち
- **Done**: 完了

カンバンビューで直感的に操作できます。

### Q13: チームメンバーを招待したい

**A**: チームメンバーの招待は簡単です。

**手順**:
1. 「設定」→「チーム」→「メンバーを招待」をクリック
2. メールアドレスを入力（複数の場合はカンマ区切り）
3. 役割を選択（オーナー、管理者、メンバー、ゲスト）
4. 「招待を送信」をクリック

相手のメールに招待リンクが送信されます。リンクをクリックしてアカウントを作成すると、チームに参加できます。

### Q14: 通知の設定を変更したい

**A**: 通知は細かくカスタマイズできます。

**設定手順**:
1. 「設定」→「通知」を選択
2. 以下の項目を設定:
   - タスクの割り当て通知
   - 期限のリマインダー（1日前、1時間前など）
   - コメントの通知
   - プロジェクトの更新通知
   - メンション通知
3. 通知方法を選択（メール、プッシュ通知、デスクトップ通知）
4. 「保存」をクリック

通知を一時的にオフにしたい場合は、「おやすみモード」を使用できます。

### Q15: データをエクスポートしたい

**A**: プロジェクトデータをエクスポートできます。

**手順**:
1. プロジェクト画面で「•••」メニューをクリック
2. 「エクスポート」を選択
3. 形式を選択（CSV、JSON、PDF）
4. 「エクスポート」をクリック

ファイルがダウンロードされます。全データをエクスポートしたい場合は、「設定」→「データ」→「エクスポート」から行えます。

## AI機能

### Q16: AI機能は無料プランでも使えますか？

**A**: AI機能はProプラン以上で利用可能です。

**主なAI機能**:
- タスク自動生成
- タスク自動割り当て
- 進捗予測
- スマート通知

Freeプランでも基本的なプロジェクト管理機能はすべて利用できます。

### Q17: AIが生成したタスクが不正確です

**A**: AI精度向上のため、フィードバックをお願いします。

**フィードバック方法**:
1. AIが生成したタスクの横にある「👍」または「👎」をクリック
2. 問題点を具体的に記入（任意）
3. 送信

フィードバックはモデルの改善に使用され、将来の精度向上につながります。

また、以下を試すことで精度が向上する場合があります:
- プロジェクト説明をより詳細に記述する
- 過去の類似プロジェクトを参考として指定する
- 業種・職種の情報をプロフィールに登録する

### Q18: AIタスク自動割り当ての仕組みは？

**A**: AIは以下の要素を総合的に判断して、最適な担当者を提案します。

**考慮される要素**:
1. **スキルセット**: メンバーのプロフィールに登録されたスキル
2. **過去の実績**: 類似タスクの完了履歴
3. **現在の負荷**: 現在割り当てられているタスク数
4. **得意分野**: 過去のパフォーマンスから学習
5. **可用性**: 休暇などのスケジュール情報

最終的な担当者の決定は、プロジェクトマネージャーが行います。AI提案を参考に、手動で変更することも可能です。

### Q19: 進捗予測の精度を上げるには？

**A**: 以下の方法で精度が向上します。

1. **定期的な進捗更新**: タスクのステータスをこまめに更新
2. **実績時間の記録**: タイムトラッキング機能を活用
3. **依存関係の設定**: タスク間の依存関係を明確に
4. **過去データの蓄積**: プロジェクトを重ねるごとに精度向上

FlowMasterを使い続けることで、あなたのチームに最適化された予測モデルが構築されます。

## 統合・連携

### Q20: Slackと連携したい

**A**: Slack統合は簡単に設定できます。

**設定手順**:
1. 「設定」→「統合」→「Slack」を選択
2. 「Slackに接続」をクリック
3. ワークスペースを選択して認証
4. 通知先のチャンネルを設定
5. 通知ルールをカスタマイズ

**連携機能**:
- タスクの作成・更新をSlackに通知
- Slackからタスクのステータス変更
- `/flowmaster`コマンドでタスク検索

### Q21: GitHubと連携する方法は？

**A**: GitHub統合で開発ワークフローがスムーズになります。

**設定手順**:
1. 「設定」→「統合」→「GitHub」を選択
2. 「GitHubに接続」をクリック
3. リポジトリへのアクセスを許可
4. 連携するリポジトリを選択
5. タスクとIssueを連携

**連携機能**:
- GitHubのIssueとFlowMasterのタスクを同期
- Pull Requestの状態をタスクに自動反映
- コミットメッセージからタスク更新

### Q22: カレンダーアプリと同期できますか？

**A**: Google CalendarとOutlookカレンダーに対応しています。

**設定手順**:
1. 「設定」→「統合」→「カレンダー」を選択
2. Google CalendarまたはOutlookを選択
3. アカウント認証
4. 同期するプロジェクトを選択
5. 同期オプションを設定

**同期される情報**:
- タスクの期限
- プロジェクトのマイルストーン
- 会議・イベント

双方向同期のため、カレンダーアプリでの変更もFlowMasterに反映されます。

## モバイルアプリ

### Q23: モバイルアプリはありますか？

**A**: はい、iOS・Androidアプリを提供しています。

**ダウンロード**:
- iOS: [App Store](https://apps.apple.com/flowmaster)
- Android: [Google Play](https://play.google.com/store/apps/flowmaster)

**主な機能**:
- タスクの作成・編集・削除
- プロジェクトの閲覧
- 通知機能
- オフライン対応（ベータ版）

### Q24: モバイルアプリでオフラインでも使えますか？

**A**: オフライン機能はベータ版として提供中です。

**オフラインでできること**:
- タスクの閲覧
- タスクの編集
- 新規タスクの作成（ドラフト保存）

**制限事項**:
- 変更はオンライン復帰時に同期される
- AI機能は利用不可
- ファイルのアップロードは不可

設定で「オフラインモード」を有効にすると、データが端末にキャッシュされます。

### Q25: スマホの通知が来ません

**A**: 以下を確認してください。

**iOSの場合**:
1. 設定アプリ → 通知 → FlowMaster
2. 「通知を許可」がオンになっているか確認
3. バナー、サウンド、バッジの設定を確認

**Androidの場合**:
1. 設定 → アプリと通知 → FlowMaster
2. 通知がオンになっているか確認
3. 通知の重要度を「高」に設定

**FlowMaster側の設定**:
- 「設定」→「通知」でプッシュ通知が有効になっているか確認

## トラブルシューティング

### Q26: ページが読み込めません

**A**: 以下の手順をお試しください。

1. **インターネット接続を確認**
   - 他のWebサイトが開けるか確認

2. **ブラウザをリフレッシュ**
   - Ctrl+R（Windows）または Cmd+R（Mac）

3. **キャッシュをクリア**
   - ブラウザの設定からキャッシュと Cookie をクリア

4. **別のブラウザで試す**
   - Chrome、Firefox、Safariなど

5. **ステータスページを確認**
   - [https://status.flowmaster.tech](https://status.flowmaster.tech)でシステム状況を確認

それでも解決しない場合は、support@flowmaster.tech までお問い合わせください。

### Q27: タスクが保存されません

**A**: 以下の原因が考えられます。

**原因1: ネットワーク接続の問題**
- インターネット接続を確認
- 一時的に接続が不安定な場合、自動で再試行されます

**原因2: ストレージ上限**
- プランのストレージ容量を確認
- 必要に応じてファイルを削除またはプランをアップグレード

**原因3: ブラウザの問題**
- ブラウザのキャッシュをクリア
- 別のブラウザで試す

**対処法**:
- ページをリフレッシュせず、しばらく待つ（自動保存を試行）
- 内容をコピーして、別のテキストエディタに一時保存

### Q28: アプリが重い・遅い

**A**: パフォーマンス改善のため、以下をお試しください。

**ブラウザ版**:
1. 不要なタブを閉じる
2. ブラウザ拡張機能を無効化
3. キャッシュをクリア
4. ハードウェアアクセラレーションを有効化

**モバイルアプリ**:
1. アプリを再起動
2. キャッシュをクリア
3. アプリを最新版にアップデート
4. 端末を再起動

**根本的な対策**:
- 大規模プロジェクト（1000タスク以上）は分割を検討
- アーカイブ機能で完了したタスクを整理
- フィルタ機能で表示タスクを絞り込む

### Q29: データが消えてしまいました

**A**: FlowMasterは1日3回自動バックアップを行っています。

**復元手順**:
1. support@flowmaster.tech に連絡
2. 以下の情報を提供:
   - アカウントメールアドレス
   - データが消えた日時
   - 影響を受けたプロジェクト名
3. サポートチームがバックアップから復元

**予防策**:
- 定期的にデータをエクスポート
- 重要なプロジェクトはローカルにもバックアップ

## セキュリティ・プライバシー

### Q30: FlowMasterはセキュアですか？

**A**: はい、業界標準のセキュリティ対策を実施しています。

**セキュリティ対策**:
- **通信の暗号化**: すべての通信はHTTPS（TLS 1.3）で暗号化
- **データの暗号化**: すべてのデータはAES-256で暗号化
- **アクセス制御**: 役割ベースのアクセス管理（RBAC）
- **定期監査**: 第三者機関による定期的なセキュリティ監査
- **ISO 27001認証**: 情報セキュリティマネジメントシステムの国際規格取得

### Q31: データはどこに保存されていますか？

**A**: データは以下の場所に保存されています。

**プライマリデータセンター**: AWS 東京リージョン
**バックアップ**: AWS 大阪リージョン

すべてのデータは日本国内に保存され、GDPR・個人情報保護法に準拠しています。

### Q32: 第三者にデータは共有されますか？

**A**: お客様の許可なく第三者にデータを共有することはありません。

**例外**:
1. **法的義務**: 法令に基づく開示要求がある場合
2. **サービス提供**: 以下のサービス提供に必要な範囲で共有
   - 決済処理（Stripe）
   - メール送信（SendGrid）
   - 分析ツール（Google Analytics）- 匿名化データのみ

詳細は[プライバシーポリシー](https://flowmaster.tech/privacy)をご覧ください。

### Q33: アカウントのセキュリティを強化したい

**A**: 以下の設定を推奨します。

1. **強力なパスワード**
   - 12文字以上
   - 大文字・小文字・数字・記号を組み合わせ

2. **二段階認証（2FA）**
   - 「設定」→「セキュリティ」から有効化

3. **セッションの管理**
   - 定期的に「アクティブセッション」を確認
   - 不明なセッションは終了させる

4. **ログイン通知**
   - 新しいデバイスからのログイン時にメール通知

5. **定期的なパスワード変更**
   - 3-6ヶ月ごとに変更を推奨
