# 退屈なことは~~Python~~ AI・コーディングエージェントにやらせよう（AI・コーディングエージェントを活用した業務効率・自動化）
>~~退屈な作業に人の時間を使わないための研究...~~

## セクション概要
AIコーディングエージェント=開発者用、開発目的に使うものであるという固定観念はありませんか？
Claude
ClaudeCode（Cowork）、Antigravity、CodexといったAI・コーディングエージェントを活用して、１連の業務プロセスを自動化する方法について探求する。
具体的には、議事録作成やスプレッドシート操作・集計、ブラウザ操作の自動化、データ収集等

## 【初級～中級】 chapter-01-minutes AIを使用して、発言ログから議事録を構造化し、共有用ファイルへ変換する
この章では会議中の発言ログをもとに、議論の要点、決定事項、担当タスク、保留事項までを漏れなく整理し、実務でそのまま使える議事録として構造化する方法を扱うセクションです。
AIに対して、必要な抽出項目や出力フォーマットを明確に指示することで、発言ログから一定品質の議事録データを効率よく生成する進め方を整理します。
あわせて、作成した議事録データをExcelやWordなどの共有しやすい形式へ変換する際のツール選定、変換手順、確認観点、出力時のポイントまでを示し、作成から配布・保管までを一連の業務としてスムーズに進める方法について扱っていきます。

### 想定範囲内
- Pythonといったプログラミング文法がわからなくても受講可能です
- エージェントスキルについての知見があるとスムーズですが、無くても受講できます

### 前提条件
- AntigravityやClaudeCodeといったAIエージェントについて何となく知っている・使ったことがある
- 不明点があれば、まず自身で AI や Web 検索を活用して調べられる

### 想定範囲外
- AI や PC の操作にまだ不慣れな初心者の方
- 就職・転職活動に直結する情報やスキルの習得を目的としている方

## 【中級】chapter-02-spreadsheet　Googleスプレッドシート＋GASとAIエージェントを組み合わせてスプレッドシートを操作し、やや高度な自動化を行う
>Googleスプレッドシート＋GASとAIエージェントを組み合わせ、スプレッドシート操作とやや高度な自動化を行う

この章では、GoogleスプレッドシートとGAS、さらにAIエージェントを組み合わせて、スプレッドシートの作成や操作を自動化する方法を扱います。
自動化の実装パターンはいくつか存在するため、それぞれのメリット・デメリットを押さえながら、1つずつ順に進めていく予定です。

### 前提条件
- AntigravityやClaudeCodeといったAIエージェントについて何となく知っている・使ったことがある
- 不明点があれば、まず自身で AI や Web 検索を活用して調べられる
- エージェントスキルの基本的な概要・使い方について把握している
- PythonやJavaScriptといったプログラミング（言語は問わない）について、触れたことがある。（あるいは自身でデバック、調べることが出来る）

### 対象外
- AI や PC の操作にまだ不慣れな初心者の方
- 就職・転職活動に直結する情報やスキルの習得を目的としている方
