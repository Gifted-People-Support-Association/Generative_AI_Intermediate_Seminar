# Datasets

講義で使用するAIやシステムへの入力データを管理するディレクトリです。

## ディレクトリ構成

| ディレクトリ | 用途 |
|---|---|
| `text/` | テキストファイル（`.txt`, `.md` 等） |
| `csv/` | CSVファイル（`.csv`） |
| `json/` | JSONファイル（`.json`, `.jsonl` 等） |

## データ追加時のガイドライン

- ファイル名は内容が分かる名前を付けてください（例: `user_reviews.csv`, `product_catalog.json`）
- 個人情報や機密情報を含むデータは格納しないでください
- 大容量のデータファイル（目安: 10MB以上）を追加する場合は、Git LFS の利用を検討してください
